# 子どものスマホで使えるようにする手順（Render 版）

このアプリを **インターネット上に置いて URL を作り**、その URL を子どものスマホで開いて
「ホーム画面に追加」すると、アプリのように使えます。パスワードは付けていません（URL を知っている人だけが開けます）。

所要時間：はじめての方で **30〜40分** くらい。難しい入力はありません。

---

## 用意するもの（3つ・すべて無料）

1. **Anthropic の APIキー**（採点や面接の「頭脳」を動かす鍵）
   - https://console.anthropic.com/ で取得。`sk-ant-...` で始まる長い文字列です。
   - ※ 使った分だけ少額の料金がかかります（練習用途なら1か月で数百円〜程度）。
2. **GitHub アカウント**（コードを置く場所。無料）… https://github.com/signup
3. **Render アカウント**（アプリを動かす場所。無料）… https://render.com/ で GitHub アカウントでログインできます。

---

## ステップ1：コードを GitHub に置く

このフォルダ（`henyu-render`）の中身を、GitHub の新しい「リポジトリ」に入れます。

### いちばん簡単な方法（ブラウザだけ・コマンド不要）

1. GitHub にログイン →右上の「＋」→ **New repository**。
2. 名前を `henyu-challenge` などにして、**Private**（非公開）を選び、**Create repository**。
3. 次の画面で **「uploading an existing file」** のリンクをクリック。
4. このフォルダ（`henyu-render`）の中の **ファイルとフォルダを全部、画面にドラッグ＆ドロップ**。
   - `main.py` / `requirements.txt` / `render.yaml` / `Dockerfile` / `static` フォルダ などすべて。
5. 下の **Commit changes**（緑のボタン）を押す。これでアップロード完了です。

> パソコンの操作に慣れている場合は、このフォルダで `git init && git add -A && git commit -m "init"` →
> GitHub のリポジトリに `git push` でも構いません（私の方で git の初期化までは済ませてあります）。

---

## ステップ2：Render でデプロイする

1. https://dashboard.render.com/ を開く →左上 **New +** → **Blueprint**。
2. さきほどの GitHub リポジトリ（`henyu-challenge`）を選ぶ。
   - このフォルダに **`render.yaml`** が入っているので、Render が設定を自動で読み込みます。
3. **APIキーを入力**する画面が出ます：
   - `ANTHROPIC_API_KEY` の欄に、ステップで用意した `sk-ant-...` を貼り付け。
   - （`MODEL` は `claude-sonnet-4-6` のままで OK。もっと安く・速くしたいなら
     `claude-haiku-4-5-20251001` に変更可。）
4. **Apply**（または Create）を押すと、ビルドが始まります（数分待ちます）。
5. 完了すると、`https://henyu-challenge-xxxx.onrender.com` のような **URL** が表示されます。
   これが子どもに渡す URL です。

---

## ステップ3：子どものスマホに「アプリとして」入れる

URL を LINE などで送り、スマホの **Safari / Chrome** で開いてから：

- **iPhone（Safari）**：共有ボタン（□に↑）→「**ホーム画面に追加**」
- **Android（Chrome）**：右上「⋮」→「**アプリをインストール／ホーム画面に追加**」

ホーム画面にアイコンが出て、全画面のアプリのように起動します。

> ⚠️ LINE 内のブラウザからは「ホーム画面に追加」が出ないことがあります。
> その場合は LINE のメニューから「**外部ブラウザで開く**」を選び、Safari / Chrome で開いてください。

---

## 知っておくとよいこと

- **無料プランの「お休み」**：しばらく（約15分）使わないとアプリが休止し、次に開くとき
  **30〜50秒ほど起動を待つ**ことがあります。毎日たくさん使うなら、Render の有料プラン
  （月 $7 程度）にすると常時すぐ開きます。最初は無料で十分です。
- **料金**：Render 自体は無料枠で OK。かかるのは Anthropic の API 利用分（使った分だけ）。
- **キーは安全**：APIキーは Render の中だけに保存され、スマホ側やコードには出ません。
- **更新したくなったら**：GitHub のファイルを更新すると、Render が自動で作り直します
  （`render.yaml` の `autoDeploy: true`）。

---

## まず手元で動かして確かめたい場合（任意）

パソコンで先に動作確認できます（Mac のターミナルで）：

```bash
cd henyu-render
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...   # 自分のキー
uvicorn main:app --reload --port 8080
# → ブラウザで http://localhost:8080
```

同じ Wi-Fi のスマホから試すなら、`http://（パソコンのIPアドレス）:8080` で開けます。
